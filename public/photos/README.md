# 📷 사진 파일 안내

이 폴더에 아래 파일명으로 사진을 추가하세요.

- `photo-1.jpg` — 가장 오래된 사진 (1번째 위치)
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`
- `photo-5.jpg` — 가장 최신 사진 (5번째 위치)

> 사진을 추가한 후, `lib/constants.ts`의 `CORRECT_PHOTO_ORDER`를 실제 정답 순서에 맞게 수정하세요.
> 예: [2, 0, 4, 1, 3] → photo-3이 첫 번째, photo-1이 두 번째, ...

사진이 없어도 앱은 정상 동작합니다 (기본 placeholder UI 표시).
